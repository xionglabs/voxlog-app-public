# 需求文档

## 1. 应用概述

**应用名称：** VoxLog（口述日记）

**Slogan：** 1分钟AI口述，记录每一天

**应用定位：** 极简、本地、隐私、AI口述日记工具的Web Demo原型，用于验证交互和界面设计

**应用描述：** VoxLog是一款基于Web的口述日记原型应用，用户通过按住按钮说话的方式快速记录日常，系统自动将语音转为文字并通过AI整理成标准日记格式。应用采用本地存储方案，无需注册登录，保护用户隐私。支持日记查看、编辑、搜索、导出及AI总结功能，提供免费版、标准会员、Pro会员三种权限等级。

---

## 2. 用户与使用场景

**目标用户：**
- 希望快速记录日常生活的用户
- 重视隐私、不希望数据上传云端的用户
- 习惯口述表达、不喜欢打字的用户
- 需要AI辅助整理日记内容的用户

**核心使用场景：**
- 用户在每天结束时，通过语音快速记录当天发生的事情
- 用户在一天内多次补充记录，系统自动合并内容
- 用户查看历史日记，回顾过往记录
- 用户通过搜索功能快速找到特定日期或关键词的日记
- 用户导出日记为PDF、Word、MD或长图格式
- 用户查看AI生成的周总结、月总结或年总结

### 3.2 首页（语音输入页）

**功能说明：**
- 展示应用名称VoxLog和Slogan
- 中央显示大按钮，用户按住按钮说话，松手结束录音
- 调用浏览器Web Speech API实现语音识别，将语音转为文字
- 提供「点击输入文字」的备用输入方式
- 语音转文字完成后，调用AI整理为标准日记格式
- 当天已有日记时，AI自动合并已有内容和新内容，不覆盖、不弹窗
- 整理完成后自动保存到本地存储
- 显示当天已记录次数和剩余AI整理次数（根据会员等级）

**标准日记格式：**
- 【日期】
- 【天气】（留空）
- 【今日记录】
- 【每日新知】
- 【今日备忘】
